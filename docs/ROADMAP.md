# LifeOS — Roadmap

## Current Phase: Planning

---

## MVP (Phase 1)

### Goal: Core Planning System

**Timeline:** First functional release

**Deliverables:**

- Identity Layer (Roles, Values, Goals)
- Task System (CRUD, filtering, splitting)
- Time Blocks (scheduling)
- Weekly Planning UI (grid + task pool + big rocks)
- Daily Planning UI (timeline + daily log)
- Progress feedback

**Stack:**

- Next.js + Zustand
- Node/Express + PostgreSQL
- Docker (single droplet)

---

## Phase 2: Enhanced Weekly/Daily

### Goal: Rich Planning Experience

**Features:**

- Drag-and-drop scheduling
- Role balance visualization
- Overbooking warnings
- Streak tracking
- Habit tracker
- Weekly summaries

**Est. Start:** After MVP validated

---

## Phase 3: Future Vision

### Goal: AI + Integration

**Features:**

- AI-assisted planning suggestions
- Predictive scheduling
- External calendar sync (Google, Cal.com)
- Custom dashboards
- Mobile companion app
- Analytics

**Est. Start:** After product-market fit
